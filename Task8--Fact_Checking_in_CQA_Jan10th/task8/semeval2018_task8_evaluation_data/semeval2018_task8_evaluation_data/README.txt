This directory contains the test data for the SemEval 2019 Task 8: Fact Checking in Community Question Answering Forums.


## Contents

The test data consists of the following files:

- questions_test.xml - contains the questions data for Subtask A

- answers_test.xml - contains the answers data for Subtask B



## Submission Format

The submission format is identical to the one used for the development set. Please consult the instructions in the Starting Kit for Phase #2 Practice.



## Instructions for the Evaluation Phase

You are allowed up to 3 submissions on the test data - one official and 2 contrastive. The final ranking will be based on the official submission. 

When submitting your results, please enter a description of your submission. It should contain the following information:

  1. Team ID

  2. Team affiliation

  3. Contact information

  4. Submission: “OFFICIAL”, “CONTRASTIVE1” or “CONTRASTIVE2”

  5. System specs

  - 5.1 Core approach

  - 5.2 Supervised or unsupervised

  - 5.3 Important/interesting/novel features used

  - 5.4 Important/interesting/novel tools used

  - 5.5 Significant data pre/post-processing

  - 5.6 Other data used (outside of the provided)

  6 References (if applicable)


During the evaluation phase you will not be able to see your results, they will be visible during the post-evaluation phase. If your submission has errors, you will be able to see the error log.

The evaluation phase ends on 31 January 2019, 23:59 UTC. After this date, the submissions will be closed and the results will be released.




If you have any questions or problems, please contact the task organizers on the following email address: semeval-2019-task-8-organizers@googlegroups.com 


Good luck!
